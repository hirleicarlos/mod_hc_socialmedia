# mod_hc_socialmedia
Módulo Joomla 4/5/6 para gerenciar e exibir links de mídias sociais de forma flexível, utilizando sistema de subformulário dinâmico. Suporte a SVG, imagens, classes de ícones, controle de layout (linha/coluna) e opções avançadas de estilização. Parte do ecossistema de extensões HC.
